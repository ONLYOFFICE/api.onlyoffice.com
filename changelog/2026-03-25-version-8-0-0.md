## New features

- **Document Builder**:
  - Restructured documentation to concatenate all programming languages on single pages instead of separate pages per language.
  - Rewrote the [Builder Framework overview](/docs/document-builder/builder-framework/overview) to emphasize runtime usage patterns.
  - Added comprehensive [CLI documentation](/docs/document-builder/using-cli/overview) with improved examples.
  - Changed code examples from `.docbuilder` extension to `.js` for better IDE support.
  - Improved the [Document Builder API](/docs/docs-api/additional-api/document-builder-api) async workflow documentation with detailed response parameters.
  - Added curl command examples throughout the API documentation.

- **Plugins and Macros**:
  - Added [AI custom tools text annotations](/docs/ai/ai/custom-ai-tools) feature documentation.
  - Added new [PDF macros samples](/samples/?doctype=docs&text=macros).
  - Added [freeze panes macro example](/samples/?doctype=docs&text=macros).
  - Added [bookmark last view macro example](/docs/macros/samples/macros/document-editor/bookmark-last-view).
  - Added [OData plugin](/samples/?doctype=docs&text=plugin) to plugin examples.
  - Added Chinese translations for the entire Plugins API section.

- **Office API**:
  - Added [PDF section support](/docs/office-api/get-started/playground) in the Playground.
  - Updated the Playground with new default scripts for SSE.
  - Added comprehensive Chinese translations for Office API usage documentation.
  - Added Document type selector to the Playground toolbar for choosing between documents with blank or sample content.
  - The Playground now opens with a sample document when accessed from the Playground overview pages.

- **DocSpace**:
  - Added [MCP server use cases](/docspace/mcp-server/getting-started/) documentation.
  - Added [new samples](/samples/?doctype=docspace&text=rest+api) to API Backend documentation.
  - Updated [Plugins SDK](/docspace/plugins-sdk/get-started/) documentation with improved structure and examples.
