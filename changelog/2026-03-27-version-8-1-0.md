## New features

- **Plugins and Macros**:
  - Reworked the [Get Started](/docs/plugin-and-macros/get-started/) section with improved quick-start tutorials (#649).
  - Added [Hello World Plugin](/docs/plugin-and-macros/get-started/quick-start/hello-world-plugin) guide.
  - Added [Hello World Macro](/docs/plugin-and-macros/get-started/quick-start/hello-world-macro) guide.
  - Added [Comment Text AI Tool](/docs/plugin-and-macros/get-started/quick-start/comment-text-ai-tool) tutorial.

- **Docs API**:
  - Added comprehensive [Document Server configuration](/docs/docs-api/get-started/configuration/server-config/intro) documentation (42 files).
  - New configuration sections include ActiveMQ, RabbitMQ, Redis, AI settings, security, token, storage, WOPI integration, and more.

- **Office API**:
  - Updated PDF API documentation (ApiRun, ApiRichRun, ApiTextPr, ApiShape classes).
  - Updated Presentation API documentation.
  - Updated Spreadsheet API documentation.
