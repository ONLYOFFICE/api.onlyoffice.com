## New features

- **DocSpace**:
  - Added [MCP server](/docspace/mcp-server/getting-started/) documentation section with getting started guides, client connection instructions, tutorials, use cases, troubleshooting, distribution guides, configuration reference, tools reference, and glossary.
  - Added MCP server homepage card and sidebar navigation.
- **Playground**:
  - Added [Config Playground](/docs/docs-api/get-started/playground) for the Docs API with interactive JSON Forms editor, live document preview, and split-pane layout.
  - Added screenshots for Config Playground cards.
- **Docs API**:
  - Added new entry-point page for the DocsAPI.DocEditor class.

## Improvements
