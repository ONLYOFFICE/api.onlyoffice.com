- **Docs API**:
  - Added the [`mobileEdit`](/docs/docs-api/using-wopi/wopi-discovery#mobileEdit) WOPI action.
  - Added the [Checking PDF forms](/docs/docs-api/get-started/how-it-works/checking-pdf-forms) page.
  - Added information on how to [call editor methods](/docs/docs-api/get-started/frontend-frameworks/react#calling-editor-methods-in-the-react-component) in frameworks.
  - Added the [Key concepts](/docs/docs-api/using-wopi/key-concepts) page for WOPI.
  - Added the [Config](/docs/docs-api/using-wopi/config) page for WOPI.
  - Added [Docs API](/docs/docs-api/more-information/changelog#version-83) version 8.3.

## Improvements

- **Common**:
  - Increased build speed.
  - Renamed all files and links to kebab-case.
  - Added notifications for build and publish processes.
- **Docspace**: Added images for [JavaScript SDK samples](/docspace/javascript-sdk/samples/advanced-samples/authorization).
- **Docs API**:
  - Added a note about using a [token in the request body](/docs/docs-api/additional-api/signature/request/token-in-body) and updated the [Signature section](/docs/docs-api/additional-api/signature).
  - Added a note about [default values](/docs/docs-api/using-wopi/wopi-rest-api/checkfileinfo#required-response-properties) for WOPI.
  - Updated the [Overview page](/docs/docs-api/using-wopi/overview) for WOPI.
- **Plugins and macros**: Added new [macro samples](/docs/plugin-and-macros/samples/macro-samples).
