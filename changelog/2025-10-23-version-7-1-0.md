## New features

- **DocSpace**: Updated DocSpace version to 3.5.0.
- **Office API**: Updated the [Office API documentation](/docs/office-api/get-started/overview) for version 9.1.0.

<!-- truncate -->

- **Docs API**:
  - The [`editorConfig.customization.uiTheme`](/docs/docs-api/usage-api/config/editor/customization/customization-standard-branding#uitheme) parameter is now available for the mobile editors
  - Added the `UserCanOnlyComment` property to the [`CheckFileInfo`](/docs/docs-api/using-wopi/wopi-rest-api/checkfileinfo/) WOPI operation
  - Changed the required size of the [`logo`](/docs/docs-api/usage-api/config/editor/customization/customization-standard-branding#logoimage) customization parameter
  - Added opening for the [hml](/docs/docs-api/usage-api/config/document#filetype) format
  - Added conversion from the [pptx](/docs/docs-api/additional-api/conversion-api/conversion-tables#presentation-file-formats) format to txt
