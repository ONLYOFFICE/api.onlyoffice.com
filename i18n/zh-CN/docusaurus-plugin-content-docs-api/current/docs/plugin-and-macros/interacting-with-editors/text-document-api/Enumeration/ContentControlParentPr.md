# 内容控件父级属性

内容控件父级属性。

## 属性

| 名称   | 类型   | 说明                             |
| ------ | ------ | -------------------------------- |
| Count  | number | 父对象中的元素数量。             |
| Parent | object | 内容控件的父级。例如，oParagraph。 |
| Pos    | number | 内容控件在父对象中的位置。       |
## 类型

Object（对象）



## 示例

```javascript
let contentControlParentPr = {
    "Parent" : oParagraph,
    "Pos" : 0,
    "Count" : 1
};
```
