# GetSelectedOleObjects

返回所选 OLE 对象的数组。

## 语法

```javascript
expression.GetSelectedOleObjects();
```

`expression` - 表示 [Api](Methods.md) 类的变量。

## 参数

此方法没有任何参数。

## 返回值

[OLEProperties](../Enumeration/OLEProperties.md)[]

## 示例

```javascript
window.Asc.plugin.executeMethod ("GetSelectedOleObjects");
```
