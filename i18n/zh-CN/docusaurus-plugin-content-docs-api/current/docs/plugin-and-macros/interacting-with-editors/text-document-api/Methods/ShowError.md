# 错误显示

显示错误或警告信息。

## 语法

```javascript
expression.ShowError(error, level);
```

`expression` - 表示 [Api](Methods.md) 类的变量。

## 参数

| **名称** | **必填/选填** | **数据类型** | **默认值** | **描述** |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| error | 必填 | string |  | 错误文本。 |
| level | 必填 | number |  | 错误或警告的等级，-1 表示错误，0 表示警告。 |

## 返回值

此方法不返回任何值。
