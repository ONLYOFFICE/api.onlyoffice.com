| ---- | ---- | ----------- |
| all | object | 将为所有插件设置的参数（&#123; "all" : &#123; key, value &#125; &#125;）。 |
| plugin_guid | object | 将为特定插件设置的参数。必须使用 asc.&#123;UUID&#125; 类型的插件 GUID 指定插件（&#123; "plugin_guid" : &#123; keyForSpecificPlugin : valueForSpecificPlugin &#125; &#125;）。 |
