# 跳转当前幻灯片

通过幻灯片索引设置编辑器中的当前幻灯片。

## 语法

```javascript
expression.GoToSlide(slideNumber);
```

一个表示 [Api](Methods.md) 类的变量。

## 参数

| **名称**     | **必填/选填** | **数据类型** | **默认值** | **描述**   |
| ------------ | ------------ | ------------ | ---------- | ---------- |
| slideNumber  | 必填         | number       |            | 幻灯片索引 |

## 返回值

此方法不返回任何值。
