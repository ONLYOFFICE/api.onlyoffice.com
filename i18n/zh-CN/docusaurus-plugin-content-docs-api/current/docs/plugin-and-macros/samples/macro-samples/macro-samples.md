# 宏示例

以下示例将向您展示如何使用 ONLYOFFICE 宏，并将 JavaScript 代码与 VBA 代码进行对比，以便您了解差异，并掌握如何将 VBA 代码转换为 ONLYOFFICE 宏。

## 运行示例宏

要设置并运行示例宏，请按照以下步骤操作：

1. 打开 ONLYOFFICE 桌面编辑器。
2. 在顶部工具栏中进入**视图**选项卡。
3. 点击**宏**按钮以打开宏编辑器。
4. 复制示例宏代码。
5. 将其粘贴到宏编辑器的工作区中。
6. 点击![播放图标](/assets/images/plugins/play.svg)来执行宏。
