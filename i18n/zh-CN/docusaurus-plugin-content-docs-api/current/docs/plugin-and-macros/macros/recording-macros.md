# 录制宏

ONLYOFFICE 编辑器允许您录制宏。录制宏会捕捉您在编辑器中的操作、点击和移动，并将其保存为宏以实现流程自动化。

## 录制宏的步骤

要录制宏：

1. 导航至**视图**标签页。

2. 在**宏**部分，点击**录制宏**以开始录制。如需暂停录制，可点击**暂停录制**，准备就绪后点击**继续录制**以恢复录制。

    ![录制宏](/assets/images/plugins/record-macros.png#gh-light-mode-only)
    ![录制宏](/assets/images/plugins/record-macros.dark.png#gh-dark-mode-only)

3. 完成所有必要步骤的录制后，点击**停止录制**以结束录制并保存宏。生成的宏代码将保存在**宏**窗口中。

    ![宏窗口](/assets/images/plugins/macro-window.png#gh-light-mode-only)
    ![宏窗口](/assets/images/plugins/macro-window.dark.png#gh-dark-mode-only)

4. 检查代码并根据需要调整。

    > 虽然录制宏可以生成完整且可用的宏代码，但其准确性可能有限。请务必仔细检查输出结果并进行全面测试，尤其是对于复杂的宏。

5. 点击![运行图标](/assets/images/plugins/play.svg)运行脚本。
