# 更新日志

ONLYOFFICE 桌面编辑器 API 的变更列表。

## 版本 8.1

- 向 *AscDesktopEditor* 对象添加了 [getViewportSettings](../usage-api/adding-a-dms-provider/getting-viewport-settings.md) 方法。
- 向 *AscDesktopEditor* 对象添加了 [onViewportSettingsChanged](../usage-api/adding-a-dms-provider/getting-viewport-settings.md) 事件。

## 版本 7.5

- 向 *execCommand* 方法添加了 [portal:uitheme](../usage-api/adding-a-dms-provider/changing-a-theme.md) 命令。
- 向 *portal:login* 命令添加了 [uiTheme](../usage-api/adding-a-dms-provider/login-and-logout.md#uiTheme) 参数。

## 版本 7.1

- 添加了 *--ascdesktop-support-debug-info-keep* 标志用于 [运行](../usage-api/debugging/running-in-debug-mode-on-windows.md#running-the-application) 应用程序。

## 版本 7.0

- 向带有 DMS 集成参数的配置添加了 [extraLogout](../usage-api/adding-a-dms-provider/adding-a-dms-provider.md#extraLogout) 参数。

## 版本 6.4

- 增加了在带有集成参数的 [配置](../usage-api/adding-a-dms-provider/adding-a-dms-provider.md) 中为每种 UI 主题类型使用图标的可能性。

## 版本 6.3

- 向带有 DMS 集成参数的配置添加了 [entryPage](../usage-api/adding-a-dms-provider/adding-a-dms-provider.md#entryPage) 参数。
- 向 *cloudCryptoCommand* 函数添加了 *updateEncryptionKeys* [命令类型](../usage-api/adding-a-dms-provider/encryption/encryption.md#type)。

## 版本 6.2

- 添加了带有 DMS 集成参数的 [配置](../usage-api/adding-a-dms-provider/adding-a-dms-provider.md)。
- 添加了 [execCommand](../usage-api/adding-a-dms-provider/execcommand.md) 方法。
- 添加了用于 [文档打开](../usage-api/adding-a-dms-provider/opening-documents.md) 的命令。

## 版本 6.1

- 向 userAgent 添加了 [AscDesktopEditor](../usage-api/adding-a-dms-provider/configuring-the-interaction.md) 标签。

## 版本 5.6

- 添加了 [加密](../usage-api/adding-a-dms-provider/encryption/encryption.md) 功能。
