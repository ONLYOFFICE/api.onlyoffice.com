# RelFromH

用于计算对象相对水平定位的基准的可能值。

## 类型

枚举

## 值

- "character"
- "column"
- "leftMargin"
- "rightMargin"
- "margin"
- "page"


## 示例

This example sets horizontal aligment for a drawing object.

```javascript editor-docx
// How to align a drawing to the center.

// Set a drawing horizontal aligment.

drawing.SetHorAlign("page", "center");
```
