# WatermarkDirection

水印方向。

## 类型

枚举

## 值

- "horizontal"
- "clockwise45"
- "counterclockwise45"
- "clockwise90"
- "counterclockwise90"

## 示例

设置文档中水印的方向。

```javascript editor-docx
// How to set a watermark direction.

// Set direction in watermark settings to "clockwise45".

watermarkSettings.SetDirection("clockwise45");
```
