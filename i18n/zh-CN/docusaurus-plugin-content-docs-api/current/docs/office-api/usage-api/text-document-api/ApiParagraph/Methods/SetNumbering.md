# SetNumbering

指定当前段落引用当前文档中的编号定义实例。

## 语法

```javascript
expression.SetNumbering(oNumberingLevel);
```

`expression` - 表示 [ApiParagraph](../ApiParagraph.md) 类的变量。

## 参数

| **名称** | **必需/可选** | **数据类型** | **默认值** | **描述** |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| oNumberingLevel | 必需 | [ApiNumberingLevel](../../ApiNumberingLevel/ApiNumberingLevel.md) |  | 将用于为段落分配编号的编号级别。 |

## 返回值

boolean
