# highlightColor

可用的高亮颜色。

## 类型

枚举

## 值

- "black"
- "blue"
- "cyan"
- "green"
- "magenta"
- "red"
- "yellow"
- "white"
- "darkBlue"
- "darkCyan"
- "darkGreen"
- "darkMagenta"
- "darkRed"
- "darkYellow"
- "darkGray"
- "lightGray"
- "none"


## 示例

This example sets a paragraph highlight to green.

```javascript editor-docx
// How to change a highlight color to green.

// Use a green color to highlight the ApiParagraph object.

paragraph.SetHighlight("green");
```
