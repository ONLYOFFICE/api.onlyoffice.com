# ApiFormRoles

表示 ApiFormRoles 类。


## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [Add](./Methods/Add.md) | boolean | 添加新的表单角色。 |
| [GetAllRoles](./Methods/GetAllRoles.md) | string[] | 列出所有可用角色。 |
| [GetCount](./Methods/GetCount.md) | number | 返回表单角色的数量。 |
| [GetRoleColor](./Methods/GetRoleColor.md) | null \| Object | 返回指定角色的 RGB 颜色。 |
| [HaveRole](./Methods/HaveRole.md) | boolean | 检查是否存在具有指定名称的角色。 |
| [MoveDown](./Methods/MoveDown.md) | boolean | 在填写顺序中将角色向下移动。 |
| [MoveUp](./Methods/MoveUp.md) | boolean | 在填写顺序中将角色向上移动。 |
| [Remove](./Methods/Remove.md) | boolean | 移除具有指定名称的角色。 |
| [SetRoleColor](./Methods/SetRoleColor.md) | boolean | 设置指定角色的颜色。 |
