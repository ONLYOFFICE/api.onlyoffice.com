# ApiRichContent

表示 ApiRichContent 类。


## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [AddElement](./Methods/AddElement.md) | boolean | 使用其在富文本内容中的位置添加富文本段落。 |
| [GetClassType](./Methods/GetClassType.md) | "richContent" | 返回 ApiRichContent 类的类型。 |
| [GetContent](./Methods/GetContent.md) | [ApiRichParagraph](../ApiRichParagraph/ApiRichParagraph.md)[] | 返回当前富文本内容对象中的富文本段落数组。 |
| [GetCurrentParagraph](./Methods/GetCurrentParagraph.md) | [ApiRichParagraph](../ApiRichParagraph/ApiRichParagraph.md) | 返回光标所在的当前段落。 |
| [GetCurrentRun](./Methods/GetCurrentRun.md) | [ApiRichRun](../ApiRichRun/ApiRichRun.md) | 返回光标所在的当前文本域。 |
| [GetElement](./Methods/GetElement.md) | [ApiRichParagraph](../ApiRichParagraph/ApiRichParagraph.md) | 根据其在内容中的位置返回富文本段落。 |
| [Push](./Methods/Push.md) | boolean | 将富文本段落推送到富文本内容。 |
