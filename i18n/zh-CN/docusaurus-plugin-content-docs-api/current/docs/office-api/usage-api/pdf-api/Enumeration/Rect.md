# Rect

以元组表示的轴对齐矩形。

不变量：
 - rect[0] &lt; rect[2] (x1 &lt; x2)
 - rect[1] &lt; rect[3] (y1 &lt; y2)

## 类型

对象

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| 0 | [pt](../Enumeration/pt.md) | x1（左） |
| 1 | [pt](../Enumeration/pt.md) | y1（上） |
| 2 | [pt](../Enumeration/pt.md) | x2（右） |
| 3 | [pt](../Enumeration/pt.md) | y2（下） |
