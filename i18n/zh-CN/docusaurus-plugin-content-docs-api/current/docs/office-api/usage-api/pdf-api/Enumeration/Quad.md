# Quad

以顶点平面元组表示的四边形。
顶点顺序固定：
 · 左上 → 右上 → 左下 → 右下

不变量：
 · x1 &lt;= x2（上边从左到右）
 · x3 &lt;= x4（下边从左到右）
 · y1 &lt;= y3（左边从上到下）
 · y2 &lt;= y4（右边从上到下）

## 类型

对象
