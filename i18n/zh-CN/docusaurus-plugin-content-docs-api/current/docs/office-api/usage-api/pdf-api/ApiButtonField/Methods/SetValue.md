# SetValue

为所有按钮字段控件设置图像

:::note\
此功能仅在 ONLYOFFICE Docs 付费版本中可用。\
:::

## 语法

```javascript
expression.SetValue(imageUrl);
```

`expression` - 表示 [ApiButtonField](../ApiButtonField.md) 类的变量。

## 参数

| **名称** | **必需/可选** | **数据类型** | **默认值** | **描述** |
| ------------- | ------------- | ------------- | ------------- | ------------- |
| imageUrl | 必需 | string |  | 要为按钮设置的图像 URL。 |

## 返回值

boolean
