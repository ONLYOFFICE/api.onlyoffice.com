# TabStop

段落制表位。

## 类型

Object

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Leader | "none" \| "dot" \| "heavy" \| "hyphen" \| "middleDot" \| "underscore" | 制表符前导字符。 |
| Pos | [twips](../Enumeration/twips.md) | 制表位位置，以磅的二十分之一（1/1440 英寸）为单位。 |
| Val | [TabJc](../Enumeration/TabJc.md) | 制表位对齐样式。 |
