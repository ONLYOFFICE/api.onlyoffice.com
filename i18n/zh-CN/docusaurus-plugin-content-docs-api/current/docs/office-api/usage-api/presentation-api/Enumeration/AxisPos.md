# AxisPos

图表中的坐标轴位置。

## 类型

枚举

## 值

- "top"
- "bottom"
- "right"
- "left"


## 示例

This example sets the axis values to the top position of the chart.

```javascript editor-pptx
// How to change the axis labels format of the chart.

// Set axis values to the top.

chart.SetAxieNumFormat("top", "0.00");
```
