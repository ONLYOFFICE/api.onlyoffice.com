# ApiTheme

表示 ApiTheme 类。


## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [GetClassType](./Methods/GetClassType.md) | "theme" | 返回 ApiTheme 类的类型。 |
| [GetColorScheme](./Methods/GetColorScheme.md) | [ApiThemeColorScheme](../ApiThemeColorScheme/ApiThemeColorScheme.md) | 返回当前主题的配色方案。 |
| [GetFontScheme](./Methods/GetFontScheme.md) | [ApiThemeFontScheme](../ApiThemeFontScheme/ApiThemeFontScheme.md) | 返回当前主题的字体方案。 |
| [GetFormatScheme](./Methods/GetFormatScheme.md) | [ApiThemeFormatScheme](../ApiThemeFormatScheme/ApiThemeFormatScheme.md) | 返回当前主题的格式方案。 |
| [GetMaster](./Methods/GetMaster.md) | [ApiMaster](../ApiMaster/ApiMaster.md) \| null | 返回当前主题的幻灯片母版。 |
| [SetColorScheme](./Methods/SetColorScheme.md) | boolean | 为当前演示文稿主题设置配色方案。 |
| [SetFontScheme](./Methods/SetFontScheme.md) | boolean | 为当前演示文稿主题设置字体方案。 |
| [SetFormatScheme](./Methods/SetFormatScheme.md) | boolean | 为当前演示文稿主题设置格式方案。 |
