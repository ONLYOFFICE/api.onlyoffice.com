# ApiCustomXmlPart

表示 ApiCustomXmlPart 类。


## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [Delete](./Methods/Delete.md) | boolean | 从自定义 XML 管理器中删除 XML。 |
| [DeleteAttribute](./Methods/DeleteAttribute.md) | boolean | 从指定 XPath 的 XML 节点删除属性。 |
| [DeleteElement](./Methods/DeleteElement.md) | boolean | 删除指定 XPath 的 XML 元素。 |
| [GetAttribute](./Methods/GetAttribute.md) | string \| null | 返回指定 XPath 的 XML 节点的属性。 |
| [GetClassType](./Methods/GetClassType.md) | "customXmlPart" | 返回 ApiCustomXmlPart 类的类型。 |
| [GetId](./Methods/GetId.md) | string | 返回自定义 XML 部件的 ID。 |
| [GetNodes](./Methods/GetNodes.md) | [ApiCustomXmlNode](../ApiCustomXmlNode/ApiCustomXmlNode.md)[] | 根据提供的 XPath 从自定义 XML 检索节点。 |
| [GetXml](./Methods/GetXml.md) | string | 从自定义 XML 部件检索 XML 字符串。 |
| [InsertAttribute](./Methods/InsertAttribute.md) | boolean | 将属性插入指定 XPath 的 XML 节点。 |
| [InsertElement](./Methods/InsertElement.md) | boolean | 在指定 XPath 插入 XML 元素。 |
| [UpdateAttribute](./Methods/UpdateAttribute.md) | boolean | 更新指定 XPath 的 XML 节点的属性。 |
| [UpdateElement](./Methods/UpdateElement.md) | boolean | 更新指定 XPath 的 XML 元素。 |
