# ApiIconCriterion

表示 ApiIconCriterion 类。

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Icon | [XlIcon](../Enumeration/XlIcon.md) | 返回或设置图标条件的图标。 |
| Index | number | 返回图标条件在集合中的索引。 |
| Operator | string | 返回或设置图标条件的比较运算符。 |
| Type | [XlConditionValueTypes](../Enumeration/XlConditionValueTypes.md) | 返回或设置图标条件的条件值类型。 |
| Value | string \| number | 返回或设置图标条件的阈值。 |
