# ApiAreas

表示 ApiAreas 类。

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Count | number | 返回表示集合中对象数量的值。 |
| Parent | [ApiRange](../ApiRange/ApiRange.md) | 返回指定集合的父对象。 |

## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [GetCount](./Methods/GetCount.md) | number | 返回表示集合中对象数量的值。 |
| [GetItem](./Methods/GetItem.md) | [ApiRange](../ApiRange/ApiRange.md) | 按 ID 从集合中返回单个对象。 |
| [GetParent](./Methods/GetParent.md) | number | 返回指定集合的父对象。 |
