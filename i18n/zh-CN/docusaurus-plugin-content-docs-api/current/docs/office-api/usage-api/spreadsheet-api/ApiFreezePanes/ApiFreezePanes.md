# ApiFreezePanes

表示 ApiFreezePanes 类。


## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [FreezeAt](./Methods/FreezeAt.md) | 无 | 设置活动工作表视图中的冻结单元格。提供的范围对应于将在最上方和最左侧窗格中冻结的单元格。 |
| [FreezeColumns](./Methods/FreezeColumns.md) | 无 | 冻结当前工作表的第一列或多列。 |
| [FreezeRows](./Methods/FreezeRows.md) | 无 | 冻结当前工作表的首行或多行。 |
| [GetLocation](./Methods/GetLocation.md) | [ApiRange](../ApiRange/ApiRange.md) \| null | 返回描述活动工作表视图中冻结单元格的范围。 |
| [Unfreeze](./Methods/Unfreeze.md) | 无 | 移除当前工作表中的所有冻结窗格。 |
