# PivotFieldSubtotals

分类汇总数据透视字段类型（分类汇总函数）。

## 类型

对象

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Average | boolean | 指定是否使用 AVERAGE 函数。 |
| Count | boolean | 指定是否使用 COUNTA 函数。 |
| CountNumbers | boolean | 指定是否使用 COUNT 函数。 |
| Max | boolean | 指定是否使用 MAX 函数。 |
| Min | boolean | 指定是否使用 MIN 函数。 |
| Product | boolean | 指定是否使用 PRODUCT 函数。 |
| StdDev | boolean | 指定是否使用 STDEV 函数。 |
| StdDevP | boolean | 指定是否使用 STDEV.P 函数。 |
| Sum | boolean | 指定是否使用 SUM 函数。 |
| Var | boolean | 指定是否使用 VAR 函数。 |
| VarP | boolean | 指定是否使用 VAR.P 函数。 |
