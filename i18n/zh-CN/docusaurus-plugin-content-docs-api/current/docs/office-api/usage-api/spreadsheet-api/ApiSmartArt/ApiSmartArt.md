| [GetName](./Methods/GetName.md) | string | 返回当前绘图的名称。 |

| [SetName](./Methods/SetName.md) | boolean | 设置当前绘图的名称。 |

| [SetSize](./Methods/SetSize.md) | 无 | 设置对象（图像、形状、图表）边界框的大小。 |
