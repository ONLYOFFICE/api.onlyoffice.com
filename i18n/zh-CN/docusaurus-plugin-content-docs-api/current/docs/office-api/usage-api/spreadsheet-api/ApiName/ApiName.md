# ApiName

表示 ApiName 类。

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Name | string | 为活动工作表设置名称。 |
| RefersTo | string | 返回或设置名称定义引用的公式。 |
| RefersToRange | [ApiRange](../ApiRange/ApiRange.md) | 按引用返回 ApiRange 对象。 |

## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [Delete](./Methods/Delete.md) | 无 | 删除 DefName 对象。 |
| [GetName](./Methods/GetName.md) | string | 返回 ApiName 类的类型。 |
| [GetRefersTo](./Methods/GetRefersTo.md) | string | 返回名称定义引用的公式。 |
| [GetRefersToRange](./Methods/GetRefersToRange.md) | [ApiRange](../ApiRange/ApiRange.md) | 按名称返回 ApiRange 对象。 |
| [SetName](./Methods/SetName.md) | boolean | 设置表示对象名称的字符串值。 |
| [SetRefersTo](./Methods/SetRefersTo.md) | 无 | 设置名称定义引用的公式。 |
