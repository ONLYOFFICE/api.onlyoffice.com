# ApiCharacters

表示 ApiCharacters 类。

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Caption | string | 指定字符范围的文本。 |
| Count | number | 集合中的字符数量。 |
| Font | [ApiFont](../ApiFont/ApiFont.md) | 指定字符的字体。 |
| Parent | [ApiRange](../ApiRange/ApiRange.md) | 指定字符的父对象。 |
| Text | string | 表示指定字符范围文本的字符串值。 |

## 方法

| 方法 | 返回值 | 描述 |
| ------ | ------- | ----------- |
| [Delete](./Methods/Delete.md) | 无 | 删除 ApiCharacters 对象。 |
| [GetCaption](./Methods/GetCaption.md) | string | 返回表示指定字符范围文本的字符串值。 |
| [GetCount](./Methods/GetCount.md) | number | 返回表示集合中对象数量的值。 |
| [GetFont](./Methods/GetFont.md) | [ApiFont](../ApiFont/ApiFont.md) | 返回表示指定字符字体的 ApiFont 对象。 |
| [GetParent](./Methods/GetParent.md) | [ApiRange](../ApiRange/ApiRange.md) | 返回指定字符的父对象。 |
| [GetText](./Methods/GetText.md) | string | 返回指定字符范围的文本。 |
| [Insert](./Methods/Insert.md) | 无 | 插入字符串以替换指定的字符。 |
| [SetCaption](./Methods/SetCaption.md) | 无 | 设置表示指定字符范围文本的字符串值。 |
| [SetText](./Methods/SetText.md) | 无 | 设置指定字符的文本。 |
