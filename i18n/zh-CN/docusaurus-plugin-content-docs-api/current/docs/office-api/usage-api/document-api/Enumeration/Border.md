# Border

边框属性对象。

## 类型

Object

## 属性

| 名称 | 类型 | 描述 |
| ---- | ---- | ----------- |
| Color | [ApiColor](../ApiColor/ApiColor.md) | 边框颜色。 |
| Size | [pt_8](../Enumeration/pt_8.md) | 以磅的八分之一为单位测量的边框宽度。 |
| Space | [pt](../Enumeration/pt.md) | 从文本到边框的间距偏移，以磅为单位。 |
| Type | [BorderType](../Enumeration/BorderType.md) | 边框样式。 |
