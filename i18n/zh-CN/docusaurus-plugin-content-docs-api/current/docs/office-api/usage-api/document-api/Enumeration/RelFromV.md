# RelFromV

用于计算对象相对垂直定位的基准的可能值。

## 类型

枚举

## 值

- "bottomMargin"
- "insideMargin"
- "topMargin"
- "margin"
- "outsideMargin"
- "page"
- "line"
- "paragraph"

## 示例

为绘图对象设置垂直对齐方式。

```javascript editor-docx
// How to align a drawing to the center.

// Set a drawing vertical aligment.

drawing.SetVerAlign("page", "center");
```
