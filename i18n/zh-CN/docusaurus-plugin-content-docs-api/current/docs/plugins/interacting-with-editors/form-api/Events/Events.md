# 事件

| 事件 | 描述 |
| ----- | ----------- |
| [onChangeRestrictions](./onChangeRestrictions.md) | 当编辑器中的限制更改时调用的函数。 |
| [onClick](./onClick.md) | 用户点击元素时调用的函数。 |
| [onDocumentContentReady](./onDocumentContentReady.md) | 当文档完全加载时调用的函数。 |
| [onEnableMouseEvent](./onEnableMouseEvent.md) | 用于开启/关闭鼠标或触控板事件的函数。 |
| [onExternalMouseUp](./onExternalMouseUp.md) | 当鼠标按钮在插件 iframe 外部释放时调用的函数。 |
| [onSubmitForm](./onSubmitForm.md) | 用户点击"完成并提交"按钮时调用的函数。 |
| [onTargetPositionChanged](./onTargetPositionChanged.md) | 当编辑器中的目标位置发生更改时调用的函数。 |
