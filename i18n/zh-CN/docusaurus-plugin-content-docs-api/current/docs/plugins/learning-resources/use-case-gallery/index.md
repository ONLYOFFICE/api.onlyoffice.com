# Use case gallery

**Goal:** See how samples, macros, and AI tools combine into complete solutions for real-world scenarios.

Each use case describes the scenario, the tools involved, and how they work together. Follow the links to the individual component samples to build the full solution.

## Business automation

Automate document-heavy workflows to reduce manual effort and errors.

### Invoice generation pipeline

**Scenario:** A team needs to generate, calculate, and send invoices without leaving the editor.

**Solution:**

1. Use the [Invoices plugin](/docs/plugins/samples/invoices.md) to pull order data and fill a template.
2. Apply spreadsheet macros to calculate totals, taxes, and discounts.
3. Use PDF editor macros to set validation rules before sending.

**Result:** A repeatable, one-click invoice generation workflow.

### Document package creator

**Scenario:** A sales team needs to generate contracts, offers, and reports together from a single data source.

**Solution:**

1. Use the [Templates plugin](/docs/plugins/samples/templates.md) to select and populate document templates.
2. Apply text document macros to update styles, headers, and footers automatically.
3. Export the resulting package to PDF.

## Educational tools

Build interactive language-learning and study environments inside the editor.

### Language learning suite

**Scenario:** A teacher wants to create language exercises, glossaries, and reading questions directly in a document.

**Solution:**

1. Combine [Translator](/docs/plugins/samples/translator.md), [Thesaurus](/docs/plugins/samples/thesaurus.md), and [Speech](/docs/plugins/samples/speech.md) plugins for vocabulary work.
2. Use text document macros to generate reading-comprehension questions from selected passages.
3. Use custom AI tools to produce differentiated exercises for different proficiency levels.

## Content creation

Build rich content environments for writers, editors, and designers.

### AI-assisted authoring environment

**Scenario:** A content team needs drafting, image editing, and rich formatting tools in one place.

**Solution:**

1. Use the [Photo editor plugin](/docs/plugins/samples/photo-editor.md) for inline image editing.
2. Add the [Symbol table plugin](/docs/plugins/samples/symbol-table.md) for special characters and typography.
3. Use the [AI plugin](/docs/plugins/samples/ai.md) with custom AI tools for drafting, rewriting, and adding structure to long texts.

## Team collaboration

Keep discussions and content close together for asynchronous and remote teams.

### Document-centric communication hub

**Scenario:** A distributed team wants to discuss, annotate, and act on documents without switching to a separate chat tool.

**Solution:**

1. Use [Extended comments](/docs/plugins/samples/extended-comments.md) to add rich, threaded annotations directly in documents.
2. Integrate the [Telegram plugin](/docs/plugins/samples/telegram.md) to post document updates and comments to a team channel.
3. Use macros and custom AI tools to generate meeting notes, summaries, and action item lists from edited document content.
