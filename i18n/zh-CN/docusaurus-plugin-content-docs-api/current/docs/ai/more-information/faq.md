# 常见问题

## 支持哪些 AI 提供商？

AI 插件支持所有与 OpenAI API 兼容的提供商，包括 OpenAI、DeepSeek 等。您也可以通过[配置自定义提供商](../guides/custom-providers.md)，添加默认列表中未包含的提供商。

## 可以使用本地 AI 模型吗？

可以。您可以使用 [Ollama](../guides/configuring-ollama-with-cors.md) 连接本地运行的模型，所有处理均在您自己的机器上完成，无需将数据发送至外部服务。

## AI 插件和 AI 助手有什么区别？

**AI 插件**是通过顶部工具栏中的 **AI** 标签页提供的完整 AI 集成功能，包括聊天机器人、摘要、翻译和语法和拼写检查等功能。

**AI 助手**是一个轻量级的内联助手，可通过在编辑器中按 `Ctrl + /` 直接唤起，无需离开文档即可通过自然语言执行编辑任务。AI 助手是 AI 插件的测试版功能。详细信息请参见 [AI 助手](../guides/ai-agent.md)。

## 哪些编辑器支持 AI 插件？

AI 插件适用于全部四种 ONLYOFFICE 编辑器：文档、电子表格、演示文稿和 PDF。

## 我的文档内容会被发送给 AI 提供商吗？

只有您明确传递给 AI 功能的内容（例如用于摘要提取或翻译的选定文本）才会发送给所配置的提供商，完整的文档内容不会被自动传输。

## 如何添加默认列表中没有的提供商？

任何与 OpenAI 兼容的提供商均可通过提供基础 URL、模型版本和 API 密钥来添加。详细的操作步骤请参阅[添加自定义提供商](../guides/custom-providers.md)。

## 在哪里可以找到现成的 AI 工具示例？

按编辑器类型分类的 AI 工具示例可在[示例页面](/samples/?doctype=ai&text=ai+tools)中找到。
