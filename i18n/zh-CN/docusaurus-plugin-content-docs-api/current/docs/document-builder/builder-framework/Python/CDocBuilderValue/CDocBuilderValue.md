# CDocBuilderValue

该类仅由 ONLYOFFICE Document Builder 用于获取所调用 JS 命令的结果。它表示一个 JS 对象的包装器。

## 语法

```cpp
class CDocBuilderValue
```

| [CreateNull](CreateNull.md)           | 创建一个 null 值。                         |
