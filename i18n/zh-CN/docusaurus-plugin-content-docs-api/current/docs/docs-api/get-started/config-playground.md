# Config Playground

Config Playground 是一个交互式工具，用于配置 ONLYOFFICE 文档 API 的初始化参数。通过它可以可视化地构建和测试配置对象，并在嵌入式编辑器中实时预览效果。

## 功能

- **可视化编辑器**：通过结构化表单配置所有初始化参数。
- **JSON 编辑器**：直接编辑或复制原始配置 JSON。
- **实时预览**：在嵌入式文档编辑器中即时查看配置效果。
