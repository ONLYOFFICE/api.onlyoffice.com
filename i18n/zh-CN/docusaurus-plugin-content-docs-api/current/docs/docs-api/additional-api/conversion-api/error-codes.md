# Error codes

转换过程中可能会发生 [错误](./response.md#error)。下面您可以找到可能的错误代码及其描述。

| 错误代码 | 描述                                                                                                                                                                                                                                                                               |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| -1         | 未知错误。                                                                                                                                                                                                                                                                            |
| -2         | 转换超时错误。                                                                                                                                                                                                                                                                 |
| -3         | 转换错误。                                                                                                                                                                                                                                                                         |
| -4         | 下载要转换的文档文件时出错。                                                                                                                                                                                                                            |
| -5         | 密码错误。                                                                                                                                                                                                                                                                       |
| -6         | 访问转换结果数据库时出错。                                                                                                                                                                                                                                     |
| -7         | 输入错误。                                                                                                                                                                                                                                                                             |
| -8         | 令牌无效。                                                                                                                                                                                                                                                                            |
| -9         | 当转换器无法自动确定输出文件格式时出错。 此错误意味着客户端必须明确指定文件应转换为哪种格式（文本文档或电子表格）。它用于在 XML 类型未知的情况下将 XML 转换为 OOXML。 |
| -10        | 超出大小限制。                                                                                                                                                                                                                                                                      |
