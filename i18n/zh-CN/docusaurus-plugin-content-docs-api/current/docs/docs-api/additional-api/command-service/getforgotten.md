# getForgotten

请求一个被遗忘的文件。

## 请求示例

```json
{
  "c": "getForgotten",
  "key": "Khirz6zTPdfd7"
}
```

## 请求参数

| 参数 | 类型   | 是否必填 | 描述                                                                       |
| --------- | ------ | -------- | --------------------------------------------------------------------------------- |
| c         | string | 必填 | 命令类型。                                                         |
| key       | string | 必填 | 用于明确标识文档文件的文档标识符。 |

## 响应示例

```json
{
  "error": 0,
  "key": "Khirz6zTPdfd7",
  "url": "https://documentserver/url-to-forgotten-document.docx"
}
```

## 响应参数

| 参数 | 类型    | 是否必填 | 描述                                                                       |
| --------- | ------- | -------- | --------------------------------------------------------------------------------- |
| error     | integer | 必填 | 错误代码。                                                            |
| key       | string  | 必填 | 用于明确标识文档文件的文档标识符。 |
| url       | string  | 必填 | 被遗忘的文件的URL。                                              |
