# meta

为所有协作编辑器更新文档的元信息。

## 请求示例

``` json
{
  "c": "meta",
  "key": "Khirz6zTPdfd7",
  "meta": {
    "title": "Example Document Title.docx"
  }
}
```

## 参数

| 参数 | 类型   | 是否必填 | 描述                                                                                                                                   |
| --------- | ------ | -------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| c         | string | 必填 | 定义命令类型。                                                                                                                     |
| key       | string | 必填 | 定义文档标识符，用于明确标识文档文件。                                                             |
| meta      | object | 必填 | 定义文档的新元信息：<br /><br />**title** - 新文档名称。<br />**类型**: string<br />**是否必填**: 必填 |

## 响应示例

``` json
{
  "error": 0,
  "key": "Khirz6zTPdfd7"
}
```

## 参数

| 参数 | 类型    | 是否必填 | 描述                                                                       |
| --------- | ------- | -------- | --------------------------------------------------------------------------------- |
| error     | integer | 必填 | 定义错误代码。                                                            |
| key       | string  | 必填 | 定义文档标识符，用于明确标识文档文件。 |
