# getForgottenList

请求被遗忘文件的列表。

## 请求示例

```json
{
  "c": "getForgottenList"
}
```

## 请求参数

| 参数 | 类型   | 是否必填 | 描述               |
| --------- | ------ | -------- | ------------------------- |
| c         | string | 必填 | 命令类型。 |

## 响应示例

```json
{
  "error": 0,
  "keys": ["Khirz6zTPdfd7"]
}
```

## 响应参数

| 参数 | 类型    | 是否必填 | 描述                                                                                   |
| --------- | ------- | -------- | --------------------------------------------------------------------------------------------- |
| error     | integer | 必填 | 错误代码。                                                                       |
| keys      | string  | 必填 | 用于明确标识文档文件的文档标识符列表。 |
