# getForgottenList

请求被遗忘文件的列表。

## 请求示例

``` json
{
  "c": "getForgottenList"
}
```

## 参数

| 参数 | 类型   | 存在 | 描述               |
| --------- | ------ | -------- | ------------------------- |
| c         | string | 必填 | 定义命令类型。 |

## 响应示例

``` json
{
  "error": 0,
  "keys": ["Khirz6zTPdfd7"]
}
```

## 参数

| 参数 | 类型    | 存在 | 描述                                                                                   |
| --------- | ------- | -------- | --------------------------------------------------------------------------------------------- |
| error     | integer | 必填 | 定义错误代码。                                                                       |
| keys      | string  | 必填 | 定义用于明确标识文档文件的文档标识符列表。 |
