# Getting started

Macros are lightweight JavaScript scripts that run directly in ONLYOFFICE documents — no installation required. Write a macro once and run it anytime to automate repetitive tasks across documents, spreadsheets, and presentations.

If you prefer to start coding right away, go directly to the [quick start guides](quick-start/hello-world-macro.md) or try things out in the [interactive playground](playground.md).
