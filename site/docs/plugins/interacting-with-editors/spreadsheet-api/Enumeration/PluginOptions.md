# PluginOptions

The plugin options.

## Type

Object

## Properties

| Name | Type | Description |
| ---- | ---- | ----------- |
| all | object | The parameters which will be set for all plugins (\{ "all" : \{ key, value \} \}). |
| plugin_guid | object | The parameters which will be set for a specific plugin. The plugin must be specified with the plugin GUID of the asc.\{UUID\} type (\{ "plugin_guid" : \{ keyForSpecificPlugin : valueForSpecificPlugin \} \}). |
