# ContextMenuType

The context menu type:**None** - not used,**Target** - nothing is selected,**Selection** - text is selected,**Image** - image is selected,**Shape** - shape is selected,**OleObject** - OLE object is selected.

## Type

Enumeration

## Values

- "None"
- "Target"
- "Selection"
- "Image"
- "Shape"
- "OleObject"
