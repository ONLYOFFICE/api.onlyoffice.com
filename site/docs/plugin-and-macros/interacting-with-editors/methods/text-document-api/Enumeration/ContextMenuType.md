# ContextMenuType

The context menu type:\n-**None** - not used,\n-**Target** - nothing is selected,\n-**Selection** - text is selected,\n-**Image** - image is selected,\n-**Shape** - shape is selected,\n-**OleObject** - OLE object is selected.

## Type

Enumeration

## Values

- "None"
- "Target"
- "Selection"
- "Image"
- "Shape"
- "OleObject"
