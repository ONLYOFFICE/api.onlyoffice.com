# ApiSortFields

Represents the ApiSortFields class.

:::note\
This functionality is available in paid ONLYOFFICE Docs editions.\
:::

## Properties

| Name | Type | Description |
| ---- | ---- | ----------- |
| Count | number | Returns the number of sort fields in the collection. |
| Parent | [ApiSort](../ApiSort/ApiSort.md) | Returns the parent Sort object. |

## Methods

| Method | Returns | Description |
| ------ | ------- | ----------- |
| [Add](./Methods/Add.md) | [ApiSortField](../ApiSortField/ApiSortField.md) \| null | Adds a sort field to the collection. |
| [Add2](./Methods/Add2.md) | [ApiSortField](../ApiSortField/ApiSortField.md) \| null | Adds a sort field with subfield support for linked data types (Stocks, Geography). |
| [Clear](./Methods/Clear.md) | None | Removes all sort fields from the collection. |
| [GetCount](./Methods/GetCount.md) | number | Returns the number of sort fields. |
| [GetParent](./Methods/GetParent.md) | [ApiSort](../ApiSort/ApiSort.md) | Returns the parent Sort object. |
| [Item](./Methods/Item.md) | [ApiSortField](../ApiSortField/ApiSortField.md) \| null | Returns the sort field at the given 1-based index. |
