# ApiPivotFilters

Represents the ApiPivotFilters class.


## Methods

| Method | Returns | Description |
| ------ | ------- | ----------- |
| [Add](./Methods/Add.md) | None | Adds a new filter to the pivot field. This method is VBA-compatible and follows the PivotFilters.Add signature from Excel VBA. Supports all major filter types including label filters, value filters, top/bottom filters, and date filters. |
