# Aggregation

A numeric value that specifies which function should be used to aggregate identical time values in the timeline data range.**1** (or omitted) - AVERAGE.**2** - COUNT.**3** - COUNTA.**4** - MAX.**5** - MEDIAN.**6** - MIN.**7** - SUM.
