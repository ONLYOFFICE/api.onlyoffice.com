# RoleProperties

## Type

Object

## Properties

| Name | Type | Description |
| ---- | ---- | ----------- |
| color | string | The role color. |


## Example

Create properties and add a new role with the specified properties.

```javascript editor-forms playground

formRoles.Add("customer", rolePr);
```
