# TextFormFormat

The text field format data.

## Type

Object

## Properties

| Name | Type | Description |
| ---- | ---- | ----------- |
| type | "none" \| "digit" \| "letter" \| "mask" \| "regExp" | The format type. |
| value | string | The format value. Required for **"mask"** and **"regExp"** types. |
