# TabJc

Custom tab types.

## Type

Enumeration

## Values

- "clear"
- "left"
- "right"
- "center"


## Example

This example set tab positions at 50, 75 and 150 points with the text aligned center, left and right at each tab stop accordingly.

```javascript editor-pdf
paraPr.SetTabs([1000, 1500, 3000], ["center", "left", "right"]);
```
