| Method | Returns | Description |

| [GetTriggerType](./Methods/GetTriggerType.md) | [AnimationTriggerType](../Enumeration/AnimationTriggerType.md) | Returns the trigger type for the animation effect. |
| [MoveTo](./Methods/MoveTo.md) | boolean | Moves the animation effect to the specified position in the sequence. |
| [SetDelay](./Methods/SetDelay.md) | boolean | Sets the delay before the animation effect starts in milliseconds. |
| [SetDuration](./Methods/SetDuration.md) | boolean | Sets the duration of the animation effect in milliseconds. |
| [SetRepeatCount](./Methods/SetRepeatCount.md) | boolean | Sets the repeat count for the animation effect. |
| [SetTriggerType](./Methods/SetTriggerType.md) | boolean | Sets the trigger type for the animation effect. |
