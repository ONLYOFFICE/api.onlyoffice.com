# TabStop

A paragraph tab stop.

## Type

Object

## Properties

| Name | Type | Description |
| ---- | ---- | ----------- |
| Leader | "none" \| "dot" \| "heavy" \| "hyphen" \| "middleDot" \| "underscore" | The tab leader character. |
| Pos | [twips](../Enumeration/twips.md) | The tab stop position measured in twentieths of a point (1/1440 of an inch). |
| Val | [TabJc](../Enumeration/TabJc.md) | The tab stop alignment style. |
