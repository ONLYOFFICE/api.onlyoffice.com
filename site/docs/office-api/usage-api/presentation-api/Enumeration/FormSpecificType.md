# FormSpecificType

The specific form type.

## Type

Enumeration

## Values

- "text"
- "checkBox"
- "picture"
- "comboBox"
- "dropDownList"
- "dateTime"
- "radio"


## Example

This example returns data of all forms in the document.

```javascript editor-pptx
let formsData = doc.GetFormsData();
```
