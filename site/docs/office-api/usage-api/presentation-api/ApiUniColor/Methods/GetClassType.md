# GetClassType

Returns a type of the ApiUniColor class.

## Syntax

```javascript
expression.GetClassType();
```

`expression` - A variable that represents a [ApiUniColor](../ApiUniColor.md) class.

## Parameters

This method doesn't have any parameters.

## Returns

"uniColor"

## Example

Determine the data type of a color object in a presentation.

```javascript editor-pptx
// How do I find out what type of color object is being used in a presentation?

// Get the class name identifier for a color object in a presentation.

const presentation = Api.GetPresentation();
const slide = presentation.GetSlideByIndex(0);
slide.RemoveAllObjects();

const presetColor = Api.CreatePresetColor("lightYellow");
const gs1 = Api.CreateGradientStop(presetColor, 0);
const gs2 = Api.CreateGradientStop(Api.RGB(255, 111, 61), 100000);
const fill = Api.CreateRadialGradientFill([gs1, gs2]);
const stroke = Api.CreateStroke(0, Api.CreateNoFill());
const shape = Api.CreateShape("flowChartMagneticTape", 300 * 36000, 130 * 36000, fill, stroke);
shape.SetPosition(608400, 1267200);
shape.SetSize(300 * 36000, 130 * 36000);

const docContent = shape.GetDocContent();
const classType = presetColor.GetClassType();
const paragraph = docContent.GetElement(0);
paragraph.SetJc("left");
paragraph.AddText("Class Type = " + classType);
slide.AddObject(shape);
```
