# ApiPathCommand

Represents the ApiPathCommand class.


## Methods

| Method | Returns | Description |
| ------ | ------- | ----------- |
| [GetHR](./Methods/GetHR.md) | string \| null | Returns the height radius of the arc. |
| [GetStartAngle](./Methods/GetStartAngle.md) | string \| null | Returns the start angle of the arc. |
| [GetSweepAngle](./Methods/GetSweepAngle.md) | string \| null | Returns the sweep angle of the arc. |
| [GetType](./Methods/GetType.md) | [PathCommandType](../Enumeration/PathCommandType.md) | Returns the type of the current path command. |
| [GetWR](./Methods/GetWR.md) | string \| null | Returns the width radius of the arc. |
| [GetX](./Methods/GetX.md) | string \| null | Returns the X coordinate for the "moveTo"/"lineTo" path commands. |
| [GetX0](./Methods/GetX0.md) | string \| null | Returns the X coordinate of the first control point for the Bezier curves. |
| [GetX1](./Methods/GetX1.md) | string \| null | Returns the X coordinate of the second control point for the cubic Bezier curves. |
| [GetX2](./Methods/GetX2.md) | string \| null | Returns the X coordinate of the end point for the cubic Bezier curves. |
| [GetY](./Methods/GetY.md) | string \| null | Returns the Y coordinate for the "moveTo"/"lineTo" path commands. |
| [GetY0](./Methods/GetY0.md) | string \| null | Returns the Y coordinate of the first control point for the Bezier curves. |
| [GetY1](./Methods/GetY1.md) | string \| null | Returns the Y coordinate of the second control point for the cubic Bezier curves. |
| [GetY2](./Methods/GetY2.md) | string \| null | Returns the Y coordinate of the end point for the cubic Bezier curves. |
