# PsfFormat

PsfFormat defines the type of formatting to apply:\
\
- "zip"       — ZIP code (e.g., 12345)\
- "zip+4"     — ZIP+4 (e.g., 12345-6789)\
- "phone"     — Phone number (e.g., (123) 456-7890)\
- "ssn"       — Social Security Number (e.g., 123-45-6789)

## Type

Enumeration

## Values

- "zip"
- "zip+4"
- "phone"
- "ssn"
