# NumberSepStyle

NumberSepStyle — defines number formatting style:\
- "us"        — 1,234.56   (English style)\
- "plain"     — 1234.56    (No separators)\
- "euro"      — 1.234,56   (European style)\
- "europlain" — 1234,56    (European without separators)\
- "ch"        — 1'234.56   (Swiss style)

## Type

Enumeration

## Values

- "us"
- "plain"
- "euro"
- "europlain"
- "ch"
