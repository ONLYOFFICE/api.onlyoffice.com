# ApiRichContent

Represents the ApiRichContent class.


## Methods

| Method | Returns | Description |
| ------ | ------- | ----------- |
| [AddElement](./Methods/AddElement.md) | boolean | Adds a rich paragraph using its position in rich content. |
| [GetClassType](./Methods/GetClassType.md) | "richContent" | Returns a type of the ApiRichContent class. |
| [GetContent](./Methods/GetContent.md) | [ApiRichParagraph](../ApiRichParagraph/ApiRichParagraph.md)[] | Returns an array of rich paragraphs from the current rich content object. |
| [GetCurrentParagraph](./Methods/GetCurrentParagraph.md) | [ApiRichParagraph](../ApiRichParagraph/ApiRichParagraph.md) | Returns the current paragraph where the cursor is located. |
| [GetCurrentRun](./Methods/GetCurrentRun.md) | [ApiRichRun](../ApiRichRun/ApiRichRun.md) | Returns the current run where the cursor is located. |
| [GetElement](./Methods/GetElement.md) | [ApiRichParagraph](../ApiRichParagraph/ApiRichParagraph.md) | Returns an rich paragraph by its position in the content. |
| [Push](./Methods/Push.md) | boolean | Pushes a rich paragraph to a rich content. |
