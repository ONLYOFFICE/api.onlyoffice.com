# GetClassType

Returns a type of the ApiInkAnnotation class.

## Syntax

```javascript
expression.GetClassType();
```

`expression` - A variable that represents a [ApiInkAnnotation](../ApiInkAnnotation.md) class.

## Parameters

This method doesn't have any parameters.

## Returns

"inkAnnot"
