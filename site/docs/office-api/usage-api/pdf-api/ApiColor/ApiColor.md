Represents the ApiColor class.


## Methods

| Method | Returns | Description |

| [FromJSON](./Methods/FromJSON.md) | [ApiColor](../ApiColor/ApiColor.md) \| null | Converts the JSON object into the ApiColor object. |
| [GetClassType](./Methods/GetClassType.md) | "color" | Returns a type of the ApiColor class. |
| [GetHex](./Methods/GetHex.md) | string | Gets the HEX string representation of the color. |
| [GetRGB](./Methods/GetRGB.md) | Object | Gets the RGB components of the color. |
| [ToJSON](./Methods/ToJSON.md) | string | Converts the ApiColor object into the JSON object. |
