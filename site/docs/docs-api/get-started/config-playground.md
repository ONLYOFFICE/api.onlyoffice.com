# Config Playground

The Config Playground is an interactive tool for configuring ONLYOFFICE Docs API initialization parameters. Use it to visually build and test a configuration object and preview the result in a live embedded editor.

## Features

- **Visual editor**. Configure all initialization parameters through a structured form.
- **JSON editor**. Directly edit or copy the raw configuration JSON.
- **Live preview**. See your configuration applied instantly in an embedded document editor.
