[Dropbox](https://www.dropbox.com/features) is a cloud-based hosting platform that allows users to securely store, share and sign documents, and collaborate with teams without exiting the app. To extend its functionality, Dropbox integrates with numerous applications via its App Center.
For example, it integrates seamlessly with [ONLYOFFICE Docs](https://www.onlyoffice.com/docs) to extend its document-handling and collaboration capabilities. By integrating with the ONLYOFFICE toolbox, Dropbox users can do the following:

- Edit, review, and collaborate with team members on text documents, spreadsheets, and presentations.
- Read PDFs
- Edit form templates in DOCXF.
- Convert between different Doc formats.

## Prerequisites

- [A Dropbox account](https://www.dropbox.com/login)
- [An ONLYOFFICE Docs account](https://www.onlyoffice.com/docspace-registration)
- [ONLYOFFICE integration link](https://dropbox.onlyoffice.com/oauth/redirect)
 
## Connecting ONLYOFFICE Docs to Dropbox

1. Log in to your Dropbox account. Sign up for an account if you're a new user.
2. Navigate to the [ONLYOFFICE Dropbox connectors page](https://helpcenter.onlyoffice.com/integration/dropbox.aspx).
3. Click the [ONLYOFFICE integration link](https://dropbox.onlyoffice.com/oauth/redirect).
4. Click **Allow** to grant ONLYOFFICE Docs permission to view and edit your Dropbox files.

## How it works

After granting access to ONLYOFFICE Docs, Dropbox users can work on their office documents without leaving Dropbox by following these steps:

1. Hover on the file of interest and click the three-dots menu (⋮) to open the file context menu.
2. Select Open in → ONLYOFFICE Docs.

This opens the selected document, spreadsheet, or presentation in a new browser tab using the corresponding ONLYOFFICE editor in full-screen mode.

## Supported data formats

With the Dropbox-ONLYOFFICE Docs integration, Dropbox users can work with files in these formats:

| **Document operations**               | **Supported formats**                                                                                                                 |
|---------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| Editing                               | DOCM, DOCX, DOCXF (Saved as PDF), DOTM, POTM, PPSM, PPSX, PPTM, PPTX, XLSM, XLSX, XLTM, XLTX.                                         |
| Editing (with possible data loss)     | CSV, EPUB, FB2, HTML, ODP, ODS, ODT, OTP, OTT, OTS, RTF, TXT.                                                                         |
| Converting to Office Open XML formats | CSV, DOC, DOT, EPUB, ET, FB2, FODP, FODS, FODT, HTM, HTML, ODP, ODS, ODT, OTP, OTS, OTT, MHT, POT, PPS, PPT, RTF, TXT, XLS, XLT, XML. |
| View only                             | DJVU, OXPS, PDF, XPS.                                                                                                                 |
