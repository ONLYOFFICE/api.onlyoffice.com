# Overview

The ONLYOFFICE DocSpace JavaScript SDK lets you embed DocSpace directly inside your web application. You can integrate a full document workspace, a standalone editor, a file picker, or a background authentication frame.

The SDK works by injecting an iframe, either through an HTML script tag or using our [npm package](https://www.npmjs.com/package/@onlyoffice/docspace-plugin-sdk), into your page and establishing a postMessage channel between your application and DocSpace.

There are multiple [integration modes](), which allow you to configure and work with different types of DocSpace instances, based on your requirements.

Want to integrate DocSpace in your web application? Check the [Getting Started](../get-started/prerequisites.md) section for more details on how to get started.
