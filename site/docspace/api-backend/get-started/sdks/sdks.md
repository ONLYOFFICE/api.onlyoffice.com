# SDKs

The ONLYOFFICE DocSpace SDKs are the libraries that provide tools for integrating and managing DocSpace features within your applications. This simplifies interaction with the DocSpace backend REST API by offering ready-to-use methods and models. These SDKs support a number of popular programming languages.
