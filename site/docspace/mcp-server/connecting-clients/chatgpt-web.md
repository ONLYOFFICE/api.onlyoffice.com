#  ChatGPT Web

This instruction describes how to connect to the DocSpace MCP server using [ChatGPT Web](https://chatgpt.com).

### Remote DocSpace MCP Server

Connect to the MCP server running remotely using ChatGPT's Connectors.

1. Open ChatGPT Web;
2. Navigate to Settings;
3. Navigate to Apps;
4. Navigate to Advanced Settings;
5. Toggle on Developer Mode;
6. Navigate back to Apps;
7. Click "Create app";
8. Enter a name (e.g., "ONLYOFFICE DocSpace MCP");
9. Enter a URL (e.g., https://mcp.onlyoffice.com/mcp);
10. Select "OAuth" in Authentication;
11. Check the "I understand and want to continue" box;
12. Click "Create";
    ![Add MCP Server](/assets/images/docspace/mcp_server/chatgpt_web.png)
13. Complete the OAuth authentication process:
    - Sign in to your DocSpace account by entering your email and password and clicking "Sign In".
    - If you have more than one account associated with the entered email, choose one of them.
    - Allow the MCP Remote Server to access the specified data in your DocSpace account.
