#  Le Chat

This instruction describes how to connect to the DocSpace MCP server using [Le Chat](https://chat.mistral.ai).

### Remote DocSpace MCP Server

Connect to the MCP server running remotely using Le Chat's Connectors.

1. Open Le Chat;
2. Navigate to Intelligence;
3. Navigate to Connectors;
4. Click "Add Connector";
5. Navigate to Custom MCP Connector;
6. Enter a name (e.g., "ONLYOFFICE_DocSpace_MCP");
7. Enter a server (e.g., https://mcp.onlyoffice.com/mcp);
8. Select "OAuth2.1" in Authentication Methods;
9. Click "Connect";
![Add MCP Server](/assets/images/docspace/mcp_server/le_chat.png)
10. Complete the OAuth authentication process:
    - Sign in to your DocSpace account by entering your email and password and clicking "Sign In".
    - If you have more than one account associated with the entered email, choose one of them.
    - Allow the MCP Remote Server to access the specified data in your DocSpace account.
