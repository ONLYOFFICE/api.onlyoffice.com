# Changelog

The list of changes for ONLYOFFICE DocSpace Plugins SDK.

## Version 2.0

- Changed: separated file security into a new `FilesSecurity` enum.
- Added the *fileSecurity* and *security* properties to [IFileItem](../usage-sdk/coding-plugin/plugin-items/fileitem.md).
- Added the *withoutBodyPadding* and *withoutHeaderMargin* properties to [IModalDialog](../usage-sdk/coding-plugin/plugin-components/modaldialog.md).
- Added the *updateCreateDialogModal* action to [Actions](../usage-sdk/coding-plugin/events.md).
- Added the *isAutoFocusOnError*, *errorText*, *onError*, *onChange*, *isCloseAfterCreate*, and *isCreateDisabled* properties to [ICreateDialog](../usage-sdk/coding-plugin/plugin-components/createdialog.md).
- Added the *itemSecurity* property to [IContextMenuItem](../usage-sdk/coding-plugin/plugin-items/contextmenuitem.md).
